The Player Specification

This class models a player, a professional football player

XML version of specification written by Soren Knutson, Sunday, December 11, 2011




CLASS Player

No superclass.

State variables:

string name
The name of the player.

string position
What position the player plays as.

string output
A string of stats that will be displayed when the player's team is highlighted.

float games
The number of games the player has played.

float comp
The number of completions the player has made.

float att
The number of passing attempts the player has made.

float pyds
The number of passing yards the player has.

float inter
The number of interceptions the player has thrown.

float sack
The number of times the play has been sacked

float rush
The number of rushing attempts a player has made.

float ruyds
The number of yards a player has rushed.

float tar
The number of times the player has been passed to.

float cat
The number of catches the player has made.

float reyds
The number of receiving yards the player has.

float fumb
The number of times the player has fumbled the ball.

float fg20a
The number of field goal attempts at 20-30 yrds

float fg20m
The number of field goals made from 20-30 yards.

float fg30a
The number of field goal attempts at 30-40 yards.

float fg30m
The number of field goals made from 30-40 yards.

float fg40a
The number of field goal attempts at 40-50 yards.

float fg40m
The number of field goals made from 40-50 yards.

float fg50a
The number of field goal attempts at 50 yards and up.

float fg50m
The number of field goals made from 50 yards and up.

Constructors:

Player
Player default constructor

No arguments.

State change:
Sets all stats of the player to zero.

Player
A ifstream constructor

Arguments:

ifstream is
A reference to an ifstream

State change:
Initializes all the variables to their appropriate values. Also determines the players position and creates the appropriate string of stats for the output variable. This can be done by using two stringstreams and a string.

Destructor:
~Player
No destructor needed

State change:
none

Methods:

float commpercent
Returns the number of completions over the number of attempts.

No arguments.

State change:
none

Return value:
Returns the number of completions over the number of attempts.

float ypcomp
Yards per completion

No arguments.

State change:
none

Return value:
Returns the number of receiving yards over the number of completions.

float interrate
Interception rate

No arguments.

State change:
none

Return value:
Returns the then number of interceptions over the number of attempts

float sackrate
The percentage of time the QB is sacked

No arguments.

State change:
none

Return value:
Returns the number of sacks over the number of attempts

float ypr
Yards per rush

No arguments.

State change:
none

Return value:
Returns the number of rushing yards over the number of rush attempts

float tpg
Targets per game

No arguments.

State change:
none

Return value:
Returns the number of times the ball was thrown to a player over the number of games the player has played.

float catchrate
The catching percentage of a player.

No arguments.

State change:
none

Return value:
Returns the number of catches the player has made over the number of times the ball has be thrown to that player.

float ypcat
Yards per catch

No arguments.

State change:
none

Return value:
Returns the number of receiving yards divided by the number of catches the player has made.

float fumbrate
The rate at which a player fumbles the ball to the other team.

No arguments.

State change:
none

Return value:
Returns the number of fumbles a player has made over the summation of the completions of a QB, the number of rush attempts, and the number of sacks.

float fg20
The percentage of success a player has to make a kick at 20-30 yards.

No arguments.

State change:
none

Return value:
Returns the number of attempts at 20-30 yards over the number of times the player has made a kick at 20-30 yards.

float fg30
The percentage of success a player has to make a kick at 30-40 yards.

No arguments.

State change:
none

Return value:
Returns the number of attempts at 30-40 yards over the number of times the player has made a kick at 30-40 yards.

float fg40
The percentage of success a player has to make a kick at 40-50 yards.

No arguments.

State change:
none

Return value:
Returns the number of attempts at 40-50 yards over the number of times the player has made a kick at 40-50 yards.

float fg50
The percentage of success a player has to make a kick at 50 yards and up.

No arguments.

State change:
none

Return value:
Returns the number of attempts at 50 yards an up over the number of times the player has made a kick at 50 yards and up.
